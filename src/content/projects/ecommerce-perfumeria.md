## Descripción del Proyecto

E-commerce completo para tienda de perfumería desarrollado con Next.js 14 y Firebase. Incluye catálogo de productos, carrito de compras, gestión de inventario en tiempo real y panel administrativo para el dueño del negocio.

## Características Principales

- 🛒 **Carrito de Compras** - Persistencia con Zustand, sincronización en tiempo real
- 📦 **Gestión de Inventario** - Control de stock con alertas automáticas
- 👥 **Panel Admin** - Dashboard con métricas de ventas y gestión de productos
- 🔐 **Autenticación** - Login con Firebase Auth (Google + Email)
- 📱 **Diseño Responsivo** - Mobile-first con Tailwind CSS
- ✨ **Animaciones Suaves** - Transiciones con Framer Motion
- ☁️ **Imágenes Cloudinary** - Optimización automática de imágenes

## Arquitectura

- **Frontend:** Next.js 14 (App Router), React 18, TypeScript
- **Estilado:** Tailwind CSS con diseño responsivo
- **Estado:** Zustand para gestión global
- **Backend:** Firebase (Auth, Firestore, Storage)
- **Imágenes:** Cloudinary CDN
